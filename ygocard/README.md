# YgoCard Help

Creates YGO Card

# ygo
 - Usage: `[p]ygo <member> [atk=0] [deff=0] <skill_text> `
 - Aliases: `yugi`
 - Cooldown: `1 per 5.0 seconds`

Make a ygocard...



![Example](https://bae.lena.moe/L0DzoEnHxAFj.png)

If no attack and defense values are set, it will convert into a Skill/Trap card (Randomly)
