## Installation

<!-- So you can copy and paste it one by one :D -->

```md
[p]load downloader
```

```md
[p]repo add glas-cogs https://github.com/djtomato/glas-cogs
```

```md
[p]cog install glas-cogs <cog_name>
```

```md
[p]load <installed_cog>
```

**\*Note**: `[p] is your bot's prefix, so replace it!`
