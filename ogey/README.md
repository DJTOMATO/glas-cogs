# ogey
 - Usage: `[p]ogey `

Generates Ogey Image.

## ogey gif
 - Usage: `[p]ogey gif `
 - Aliases: `gifs`
 - Cooldown: `1 per 3.0 seconds`

Generates a random Ogey GIF.

## ogey image
 - Usage: `[p]ogey image `
 - Aliases: `images`
 - Cooldown: `1 per 3.0 seconds`

Generates a random Ogey image.
